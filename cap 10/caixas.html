explicando modelo de caixas.

é relativamente facil explicar o modelo de caixas.

Esse modelo é simplesmente um agrupamento ou aninhamento de caixas que são basicamente tags.

Existem tipos de "caixas" as box-level e as inline-level. Uma box-level é o h1 e uma inline-level é o link ou <a>

Pra ter uma base já que tu vai precisar, existem "niveis de caixa", esse é um termo horrivel mas vou usar aqui bele.

Bom pra falar de caixas temos que entender como elas são construidas. Pra aprender como elas são contruidadas vá até esse link: Mestre Guanabara.

A outra coisa que é precisa é saber configurar as caixas e fazer shorthands.
